# Boundary-Making

This repository organizes the research workflow for the comparative study of boundary-making narratives and naturalization policy in Korea and Taiwan.

## Project scope
- IV corpus: speeches, legislative debates, policy justification documents
- DV corpus: nationality laws, enforcement decrees, administrative rules
- Measurement strategy: dictionary-based analysis
- AI use: limited to dictionary expansion support, not final measurement

## Core principle
IV and DV must be kept strictly separate in both corpus construction and analysis.

## Repository structure
- `data/`: raw, interim, and processed data
- `dictionaries/`: discourse and legal-policy dictionaries
- `scripts/`: cleaning, measurement, and analysis scripts
- `outputs/`: tables, figures, validation outputs
- `docs/`: documentation
- `notes/`: working memos
